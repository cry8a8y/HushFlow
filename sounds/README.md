# HushFlow Sounds

Place sound files here for breath transition audio cues.

## Sound names
- `inhale.ogg` — played at start of inhale
- `exhale.ogg` — played at start of exhale
- `hold.ogg` — played at start of hold
- `complete.ogg` — played when session ends

## Supported formats
WAV, MP3, OGG

## Custom sounds
Place your own files in `~/.hushflow/sounds/` to override defaults.

## Requirements
One of: ffplay (FFmpeg), mpv, afplay (macOS), or paplay (PulseAudio)
